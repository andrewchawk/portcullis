# Todo App

### start local server

e.g.

#### Python

run `python3 -m http.server` and navigate to `http://localhost:8000/`

#### NodeJs

run `http-server -p 8000` and navigate to `http://localhost:8000/`

### todo-app

_http://localhost:8000/app/_

### visualize todo-app

_http://localhost:8000/visualize_dataflow/_
